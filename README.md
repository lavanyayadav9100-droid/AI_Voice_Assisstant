# AI Voice Assistant

A simple AI Voice Assistant built with Python, Gradio, and the Google Gemini API. It converts speech to text, generates AI responses, and speaks the reply using text-to-speech.

## Features
- 🎤 Voice input
- 📝 Speech-to-text
- 🤖 AI responses with Gemini
- 🔊 Text-to-speech output

## Technologies
- Python
- Gradio
- Google Gemini API
- SpeechRecognition
- gTTS

## Run

```bash
pip install -r requirements.txt
python app.py
```
